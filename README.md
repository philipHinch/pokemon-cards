 # Pokemon Cards Viewer
 
 You can search your favourite pokemon and check out their stats! 
 
 If you are undecided, let the random button choose the pokemons for you.
 
  Click the card for an awesome effect!
 
 Technologies used:
 
* HTML
* CSS
* VANILLA JAVASCRIPT
